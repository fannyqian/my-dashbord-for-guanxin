# 南区7月目标进展看板 — 使用指南

## 快速启动

**方式1：直接打开**

双击 `南区7月目标看板.html` 即可在浏览器中查看。

**方式2：更新底表后重新生成**

```bash
cd "C:\Users\fanny\Desktop\数据日日"
PYTHONIOENCODING=utf-8 python generate_dashboard.py
start 南区7月目标看板.html
```

**方式3：在对话中召唤**

在 Claude Code 对话中直接说：
- "更新看板" — 用最新底表重新生成
- "导入底表" — 我帮你更新底表文件后重新生成

## 看板功能一览

### 顶部按钮

| 按钮 | 功能 |
|---|---|
| 📥 导出底表 | 下载当前订单底表 Excel |
| 📤 导入底表 | 上传新底表 → 全量自动刷新 |
| 📸 快照 | 弹出干净摘要页（KPI+角色+分组）→ Ctrl+P 存PDF分享 |
| 📥 普陀模板 | 导出普陀门店数据模板 |
| ⚠ 导入普陀 | 上传普陀个案业绩 → 联动更新 |

### 四个标签页

| 标签 | 内容 | 特有功能 |
|---|---|---|
| 🏢 按角色 | 销售/班主任/门店 3行汇总 | — |
| 📋 按分组 | 线上5组+班主任2组+线下11店 | 列头点击排序 |
| 👤 线上销售 | 51人明细 | 🏆 TOP10排行榜 |
| 🏥 线下个案 | 60人明细 | 🏆 TOP10排行榜 |

### 交互操作

| 操作 | 方式 |
|---|---|
| ✏️ 编辑数字 | 双击单元格 → 修改 → 点别处自动刷新 |
| 🔄 排序 | 点击任意列头 → 升序↑/降序↓ |
| 🏆 排行榜 | 点击TOP10按钮 → 弹窗+日期筛选+🥇🥈🥉 |
| 📅 日期筛选 | TOP10弹窗内修改日期范围 → 点筛选 |

### 颜色标记

| 进度条颜色 | 含义 |
|---|---|
| 🟢 绿色 | 完成率 ≥ 32.3%（时间进度），正常 |
| 🟡 黄色 ⚠️ | 完成率 ≥ 16.1% 但落后时间进度，可能完不成 |
| 🔴 红色 🚫 | 完成率 < 16.1%，必定完不成，特别关注 |

| 退费标红 | 阈值 |
|---|---|
| 销售/班主任个人 | 退费率 > 30% |
| 门店CM个人 | 退费率 > 10% |

## 普陀特殊规则

- 普陀业绩不从订单底表计算，通过 ⚠导入普陀 单独更新
- 普陀模板格式：门店业绩 | 门店退款 | 个案经理 | 完成业绩
- 普陀目标：200万（杨宇航）

## 指标计算公式

| 指标 | 公式 |
|---|---|
| 完成 | 7月1-10日 pay_amount 合计 |
| 完成率 | 完成 / 目标 |
| 时间进度 | 10/31 = 32.3% |
| 趋势 | 完成率 / 时间进度 |
| 上月同期 | 6月1-10日 pay_amount 合计 |
| 完成环比 | (本月完成 - 上月完成) / 上月完成 |
| 本月退费 | 7月1-10日 back_dt 对应的 pay_amount 合计 |
| 退费率 | 退费 / 完成 |
| 净流水 | 完成 - 退费 |
