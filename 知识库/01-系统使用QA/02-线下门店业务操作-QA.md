# 线下门店业务操作 - 常见问题QA

> 最后更新：2026-03-25
> 适用场景：渠道客服、CM、门店管理员日常操作查阅

---

## 一、渠道客户录入

**Q: 如何手动新增渠道客户？**
A: 左侧菜单【渠道客户】→ 点击【新增客户】→ 完善客户信息 → 点击【确定】。

**Q: 重复手机号可以录入吗？**
A: 重复手机号**支持录入**，但该客户**无法分配**给任何人。

**Q: 如何批量导入渠道客户？**
A: 步骤如下：
1. 【渠道客户】→ 点击【批量导入】→ 点击【下载导入模版】
2. 按模版整理数据，注意：
   - **姓名、手机号、渠道来源**为必填项
   - 渠道来源填写编号（如：0001），具体编号查看模版第2个sheet【渠道来源】
   - 地区按**省/市格式**填写（如：北京/北京市），具体查看第3个sheet【地区】
3. 上传整理好的Excel文件
4. 若导入失败，点击【下载失败记录文件】，按失败原因逐条修改后再次导入（仅需重新导入失败数据）

## 二、客服跟进客户

**Q: 客服在哪里查看需要跟进的客户？**
A: 左侧菜单【中台客服】→【用户跟进】列表，可查看所有待跟进客户。

**Q: 客服有哪几种跟进操作？**
A: 共4种操作：
- **添加跟进记录**：列表点击【跟进】→ 填写跟进信息 → 点击【确定】（可设置下次跟进提醒时间，到时企微会收到通知）
- **查看用户详情**：列表点击【查看】，查看用户详情及跟进记录
- **预约到店**：见下方预约到店说明
- **预约咨询**：见下方预约咨询说明

## 三、预约到店

**Q: 如何给客户预约到店？**
A: 如果您是客服身份，中台客服列表点击【预约到店】→ 选择预约门店、预约时间 → 可直接选择CM→ 点击【确定】。
如果您是个案身份，点击前台预约找到咨询师→ 日历列表点击【预约咨询】→ 选择订单 → 完善预约信息 → 点击【确定】。

**Q: 预约时需要同时预留咨询师怎么操作？**
A: 开启"预留咨询师"开关后，填写：
- **咨询师**：仅能预留参与排班的咨询师；不参与排班的咨询师，可在预约日历中禁用其时间段达到预留效果
- **预约原因**：老客户预留 / 团辅活动预留 / 小红书/大众点评/美团预留 / 其他
- **咨询时间**：仅能预留可预约的时间段


**Q: 如何修改已预约的到店时间？**
A: 左侧菜单【前台预约 → 预约明细】→ 到店列表中找到"预约类型=到店"的记录 → 点击【修改预约时间】→ 选择新时间 → 点击【确定】。

**Q: 如何取消到店？**
A: 左侧菜单【前台预约 → 预约明细】→ 到店列表 → 点击【确认到店】后 → 点击【取消到店】。

## 四、预约咨询

**Q: 所有客户都可以预约咨询吗？**
A: 不是。**仅有咨询订单的客户**才可以预约咨询。如果无订单预约咨询且在保证客户高概率会咨询的基础上可以预留咨询师的时间，在预约日历里面操作。

**Q: 如何给客户预约咨询？**
A: 列表点击【预约咨询】→ 选择订单 → 完善预约信息 → 点击【确定】。

**Q: 哪些情况预约咨询需要咨询师确认？**
A: 以下三种情况需要咨询师确认后才会流转到【已分配】：
1. 预约24小时内的咨询
2. 咨询师从未服务过此客户
3. 预约的是咨询师非排班日程

> 注意：等待确认期间**请勿重复预约**。

**Q: 如何通过日历查看咨询师和CM的排班情况？**
A:
- **咨询师日历**：左侧菜单【预约日历】，选择门店、日期、咨询师查看排班
- **CM日历**：左侧菜单【CM日历】，选择门店和意向日期，可了解每位CM的接待情况；鼠标划过CM日程区域出现+号，可直接创建预约到店
**Q: 如何取消预约？**
A: 需要联系客户自行取消预约，所有人均不可帮客户取消预约。客户操作步骤：登录APP或者小程序，进入服务->待办，找到对应的待咨询记录，点击取消预约即可。

## 五、线索分配

**Q: 如何分配渠道客户给？**
A: 左侧菜单【公海数据 → 中台线索池】→ 可看到所有非重复渠道客户 → 点击【分配】或勾选多条点击【批量分配】→ 选择员工 → 点击【确定】。对应客服企微会收到分配通知。

**Q: 什么叫重复客户？重复客户能分配吗？**
A: 重复客户指在中台线索池及门店人员名下重复出现的客户。重复客户**无法分配**，但可通过筛选看到。

**Q: 如何分配门店公海池的客户给CM？**
A: 左侧菜单【公池数据】→ 点击管理的门店 → 找到待分配线索 → 点击【分配】或【批量分配】→ 选择员工 → 点击【确定】。对应CM可在【全部客户】菜单查看。

## 六、其他常见问题

**Q: 系统出现异常，如何清理浏览器缓存？**
A:
1. 浏览器右上角点击三个点 → 找到【历史记录】→ 点击【历史记录】
2. 点击【删除浏览数据】
3. 时间范围选择【不限时间】，勾选【Cookie及其他网站数据】和【缓存的图片和文件】
4. 点击【从设备中删除】即可
