# 前台收银系统 - 常见问题QA

> 最后更新：2026-03-25
> 适用场景：前台收银操作时查阅

---

## 一、基本操作流程

**Q: 收银操作的基本步骤是什么？**
A: 共三步：
1. **输入用户信息和售卖人信息**（同时可查询用户历史消费情况）
2. **选择用户需购买的商品**（可多选）
3. **核销券码及付款**
特别提示：收银是收钱的地方，不是录客户的路径哈，需要保证客户已经从新增客户里面录入并且与自己绑定归属关系了。
---

## 二、付款方式

**Q: 系统支持哪些付款方式？**
A: 共6种付款方式：
| 方式 | 说明 |
|------|------|
| **POS机主动扫码（客户扫码）** | 客户主动打开摄像头扫码付款，选择后发起收款，POS机提示需要付款 |
| **POS机展示付款码（扫客户码）** | 客户出示付款码，前台用POS机扫，选择后发起收款 |
| **POS机刷卡** | 用户需要刷卡时选择 |
| **银行转账** | 填写门店对公账户、转账金额和转账凭证 |
| **用户核销券码** | 核销用户持有的券码 |
| **现金** | 使用现金付款 |

**Q: 在POS机上如何发起交易？**
A: 在POS机上点击【发起交易】即可。

---

## 三、券码核销

**Q: 用户持有美团/大众点评等平台券码，如何操作？**
A: 在第3步核销券码时，先完成平台券码核销，再对剩余金额选择其他付款方式补差价。

**Q: 场景示例：用户在美团购买了1个398首次咨询，同时还要购买10次咨询套餐×2和菌群移植×3，怎么操作？**
A:
1. 第1步录入用户信息和售卖人信息
2. 第2步选择所有商品（398首次咨询 + 10次咨询套餐×2 + 菌群移植×3）
3. 第3步先核销美团券码，再对剩余金额选择对应付款方式付款

---

## 四、花呗分期免息券

**Q: 花呗分期免息券有什么用？**
A: 售卖时可为用户申请花呗分期免息券，让用户享受免息分期，降低付款门槛，促进转化。

**Q: 什么时候可以给用户申请花呗免息券？**
A: 销转过程中若用户犹豫是否购买，或用户主动询问是否支持花呗免息，均可申请。

**Q: 花呗免息券的适用条件是什么？**
A: 需满足：
- 全款支付
- 选择支付平台为**支付宝支付**
- 免息期选择**3期或6期**

**Q: 分享花呗免息券后有效期多长？**
A: 分享后（生成二维码）开始倒计时，**有效期6小时**。

**Q: 超过6小时用户没有领取或没有使用怎么办？**
A:
- 用户未领取：6小时后无法领取，需重新分享
- 用户已领取但未使用：6小时后免息券失效，无法使用，需重新分享
- 重新分享时可对用户说"重新申请"，以增加权益稀缺性和价值感

**Q: 用户侧如何使用花呗免息券？**
A: 用户领取免息券后，支付购买时选择花呗支付，系统自动提示分期免息。
