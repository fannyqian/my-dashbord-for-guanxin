# 企业微信智能问答系统 - 快速启动指南

## 环境准备

### 1. 安装依赖
```bash
pip install requests openai
```

### 2. 设置环境变量
```bash
# Windows (CMD)
set DEEPSEEK_API_KEY=your_api_key_here

# Windows (PowerShell)
$env:DEEPSEEK_API_KEY="your_api_key_here"

# Linux/Mac
export DEEPSEEK_API_KEY="your_api_key_here"
```

## 使用流程

### 方案A：自动化流程（需要API权限）

#### 第一步：获取文档内容
```bash
python fetch_docs.py
```

如果遇到权限问题（错误码48002），请参考 `权限配置指南.md` 配置企业微信应用权限。

#### 第二步：生成QA知识库
```bash
python generate_qa.py
```

### 方案B：手动导出流程（临时方案）

#### 第一步：手动导出文档
1. 打开以下文档链接：
   - https://doc.weixin.qq.com/wiki/w.AI4A4AcKABA.R2hWBEPzw3c
   - https://doc.weixin.qq.com/wiki/w.AI4A4AcKABA.Jx-CZq7P3Rs
   - https://doc.weixin.qq.com/sheet/e3_AXAAoQYrAIQCN61pqVd8uSQWwhAqa

2. 将内容复制或导出为txt格式

3. 保存到 `./raw_docs/` 目录，命名为：
   - `系统相关知识库.txt`
   - `门店个案知识库.txt`
   - `单独QA文档.txt`

#### 第二步：生成QA知识库
```bash
python generate_qa.py
```

## 常见问题

### Q: fetch_docs.py 报错 48002
A: 这是权限问题，需要在企业微信管理后台配置应用的微文档API权限。详见 `权限配置指南.md`。

### Q: generate_qa.py 报错找不到文档
A: 确保 `./raw_docs/` 目录中有 .txt 文件。可以手动导出文档放进去。

### Q: DeepSeek API 报错
A: 检查环境变量 `DEEPSEEK_API_KEY` 是否设置正确，以及API余额是否充足。

### Q: 生成的QA质量不满意
A: 可以修改 `generate_qa.py` 中的 `SYSTEM_PROMPT`，调整生成要求和风格。
